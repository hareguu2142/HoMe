적분 문제 예시

다음 적분을 계산해 보겠습니다:

$$\int_{0}^{5} \left( \lfloor x \rfloor \right)^2 \, dx$$

단계별 해설:

  1. 구간 분할:

    바닥 함수 $\lfloor x \rfloor$는 각 정수에서 값이 변합니다. 따라서 $0$부터 $5$까지 다음과 같이 구간을 나눌 수 있습니다:

    • $[0, 1): \lfloor x \rfloor = 0$
    • $[1, 2): \lfloor x \rfloor = 1$
    • $[2, 3): \lfloor x \rfloor = 2$
    • $[3, 4): \lfloor x \rfloor = 3$
    • $[4, 5): \lfloor x \rfloor = 4$
  2. 각 구간에서의 적분 계산:

    $\lfloor x \rfloor$가 각 구간에서 상수이므로, 적분은 다음과 같이 간단해집니다:

    1. $$[0, 1):$$ $$\int_{0}^{1} 0^2 \, dx = 0 \times (1 - 0) = 0$$
    2. $$[1, 2):$$ $$\int_{1}^{2} 1^2 \, dx = 1^2 \times (2 - 1) = 1$$
    3. $$[2, 3):$$ $$\int_{2}^{3} 2^2 \, dx = 2^2 \times (3 - 2) = 4$$
    4. $$[3, 4):$$ $$\int_{3}^{4} 3^2 \, dx = 3^2 \times (4 - 3) = 9$$
    5. $$[4, 5):$$ $$\int_{4}^{5} 4^2 \, dx = 4^2 \times (5 - 4) = 16$$
  3. 결과 합산:

    각 구간의 결과를 합하면:

    $0 + 1 + 4 + 9 + 16 = 30$

최종 답:

$$\int_{0}^{5} \left( \lfloor x \rfloor \right)^2 \, dx = 30$$
