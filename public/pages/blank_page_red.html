코드 스니펫 복사 기능 예제

JavaScript 코드 예시

function sumToN(n) {
    let sum = 0;
    for (let i = 1; i <= n; i++) {
        sum += i;
    }
    return sum;
}

// 예시: 1부터 100까지의 합
console.log(sumToN(100));

코드 스니펫 복사 기능 설명

이 페이지는 코드 스니펫에 복사 기능을 추가하는 방법을 보여줍니다:

  1. ClipboardJS 라이브러리를 사용하여 복사 기능을 구현했습니다.
  2. 코드 영역에 마우스를 올리면 '복사' 버튼이 나타납니다.
  3. '복사' 버튼을 클릭하면 코드가 클립보드에 복사됩니다.
  4. 복사 성공 시 알림 메시지가 표시됩니다.
  5. 복사 실패 시 오류 메시지가 표시됩니다.

이 기능은 사용자가 쉽게 코드를 복사하여 사용할 수 있게 해줍니다.
