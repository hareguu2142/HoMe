페이지 소개

이 페이지는 수학 공식과 코드 스니펫을 표시하고 복사할 수 있는 기능을 제공합니다:

  • KaTeX 라이브러리를 사용하여 수학 공식을 렌더링합니다.
  • ClipboardJS를 사용하여 코드 스니펫 복사 기능을 구현했습니다.
  • 수학 공식은 LaTeX 문법으로 작성할 수 있습니다.
  • 코드 스니펫은 복사 버튼을 통해 쉽게 복사할 수 있습니다.

아래에 수학 공식과 코드 스니펫 예시를 추가하여 사용할 수 있습니다.

수학 공식 예시

$$E = mc^2$$

코드 스니펫 예시

function helloWorld() {
    console.log("Hello, World!");
}
