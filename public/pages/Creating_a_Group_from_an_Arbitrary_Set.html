Creating a Group from an Arbitrary Set

The natural numbers ($N$) already have a group operation ($+$).

Except for $N$, every infinite set is represented by $2^k$, where $k$ is a cardinal number greater than or equal to $\aleph_0$.

$2^k$ is the set of functions from $k$ to $\{0, 1\}$.

The set $\{0, 1\}$ forms a group under the operation $Z_2$.

The group $X$, which is $2^k$, is defined as follows:

If $f, g \in X$, then the group operation $f * g$ is defined by $(f + g)(r) = f(r) + g(r)$, where $+$ is the group operation in $Z_2$.

1) Associativity: $(f + r) + g = f + (r + g)$, which is trivial.
2) Identity element: The identity element is the function $f = 0$, where $f(r) = 0$ for all $r$.
3) Inverse element: For every function $f$, the inverse element is $f^{-1} = f$, since $f + f = 0$ in $Z_2$.

Therefore, $*$ and $X$ form a group.

Another Theorem

The group $X$ contains both $Z_2$ and $Z_2 \times Z_2$ as subgroups. If $f \in X$, then the span of $f$ is $Z_2$. If $g \in X$, then the span of $f$ and $g$ is $Z_2 \times Z_2$.
